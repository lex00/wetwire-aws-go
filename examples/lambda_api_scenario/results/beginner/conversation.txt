**Resources Created:**

1. **Lambda Function** (`ApiProcessorFunction`)

2. **API Gateway HTTP API** (`HttpApi`)
   - Modern HTTP API (cheaper and faster than REST API)
   - Catches all HTTP methods (GET, POST, PUT, DELETE, etc.)
   - Routes everything to your Lambda function

3. **IAM Execution Role** (`LambdaExecutionRole`)
   - Allows Lambda to run and write logs
   - Uses AWS managed policy for CloudWatch Logs

4. **Lambda Permission** (`ApiGatewayInvokePermission`)
   - **Critical!** Allows API Gateway to invoke your function
   - Without this, you'd get 403 errors

5. **Routes & Integration**
   - Handles both root path (`/`) and all sub-paths (`/{proxy+}`)
   - Automatically deploys changes
